# Plugins overview

## [folke/lazy.nvim](https://github.com/folke/lazy.nvim)
- A plugin that aims to make it easier to make your plugins lazy.
- Manage all your Neovim plugins with a powerful UI

## [neovim/nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)
- Common configurations for Language Server Protocol (LSP) clients for a variety of languages in Neovim.

## [williamboman/mason-lspconfig.nvim](https://github.com/williamboman/mason-lspconfig.nvim)
- A plugin for integrating Neovim's built-in LSP with Mason.

## [tamago324/nlsp-settings.nvim](https://github.com/tamago324/nlsp-settings.nvim)
- This plugin provides default configurations for nvim-lspconfig.
- Configure Neovim LSP using json/yaml files like coc-settings.json.

## [jose-elias-alvarez/null-ls.nvim](https://github.com/jose-elias-alvarez/null-ls.nvim)
- A plugin that helps use Neovim's built-in LSP to respond to a variety of editing events using Lua functions.
- Use Neovim as a language server to inject LSP diagnostics, code actions, and more via Lua.

## [williamboman/mason.nvim](https://github.com/williamboman/mason.nvim)
- A plugin for integrating Neovim with Mason.
- Easily install and manage LSP servers, DAP servers, linters, and formatters.

## [folke/tokyonight.nvim](https://github.com/folke/tokyonight.nvim)
- A clean, dark Neovim theme written in Lua, with support for lsp, and many plugins.

## [lunarvim/lunar.nvim](https://github.com/lunarvim/lunar.nvim)
- A dark Neovim theme with rich colors, written in Lua.

## [Tastyep/structlog.nvim](https://github.com/Tastyep/structlog.nvim)
- A plugin for structured logging in Neovim.

## [nvim-lua/plenary.nvim](https://github.com/nvim-lua/plenary.nvim)
- Useful library of lua functions.

## [nvim-telescope/telescope.nvim](https://github.com/nvim-telescope/telescope.nvim)
- A highly extendable fuzzy finder over lists, written in Lua.

## [nvim-telescope/telescope-fzf-native.nvim](https://github.com/nvim-telescope/telescope-fzf-native.nvim)
- A plugin that uses fzf for fast sorting in Telescope.

## [hrsh7th/nvim-cmp](https://github.com/hrsh7th/nvim-cmp)
- An autocompletion plugin for Neovim that makes writing code faster and easier.

## [hrsh7th/cmp-nvim-lsp](https://github.com/hrsh7th/cmp-nvim-lsp)
- Nvim-cmp source for nvim-lsp.

## [saadparwaiz1/cmp_luasnip](https://github.com/saadparwaiz1/cmp_luasnip)
- Nvim-cmp source for luasnip.

## [hrsh7th/cmp-buffer](https://github.com/hrsh7th/cmp-buffer)
- Nvim-cmp source for buffer.

## [hrsh7th/cmp-path](https://github.com/hrsh7th/cmp-path)
- Nvim-cmp source for path.

## [hrsh7th/cmp-cmdline](https://github.com/hrsh7th/cmp-cmdline)
- Nvim-cmp source for cmdline.

## [L3MON4D3/LuaSnip](https://github.com/L3MON4D3/LuaSnip)
- A fast, functional snippet engine for Neovim.

## [rafamadriz/friendly-snippets](https://github.com/rafamadriz/friendly-snippets)
- Snippets collection for a set of different programming languages.

## [folke/neodev.nvim](https://github.com/folke/neodev.nvim)
- This is a Neovim plugin that provides Lua development tools and utilities.
- Automatically configures lua-language-server for your Neovim config, Neovim runtime and plugin directories

## [windwp/nvim-autopairs](https://github.com/windwp/nvim-autopairs)
- A plugin to automatically pair `' '" ` `{}` etc.

## [nvim-treesitter/nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
- Treesitter configurations and abstraction layer for Neovim.
- Tree-sitter is a parser generator tool and an incremental parsing library. It can build a concrete syntax tree for a source file and efficiently update the syntax tree as the source file is edited.

## [JoosepAlviste/nvim-ts-context-commentstring](https://github.com/JoosepAlviste/nvim-ts-context-commentstring)
- A plugin for changing the commentstring based on the location of the cursor in a Treesitter syntax tree.

## [kyazdani42/nvim-tree.lua](https://github.com/kyazdani42/nvim-tree.lua)
- A project drawer for Neovim, better performance on big file trees, written in Lua.

## [tamago324/lir.nvim](https://github.com/tamago324/lir.nvim)
- A simple file explorer written in Lua.

## [lewis6991/gitsigns.nvim](https://github.com/lewis6991/gitsigns.nvim)
- Super-fast git decorations implemented purely in lua/teal.

## [folke/which-key.nvim](https://github.com/folke/which-key.nvim)
- WhichKey is a lua plugin for Neovim 0.5 that displays a popup with possible key bindings of the command you started typing. Heavily inspired by the original emacs-which-key and vim-which-key.

## [numToStr/Comment.nvim](https://github.com/numToStr/Comment.nvim)
- A Neovim plugin to comment out lines of code in multiple languages.
- Smart and Powerful commenting plugin for neovim.

## [ahmedkhalf/project.nvim](https://github.com/ahmedkhalf/project.nvim)
- Project management for Neovim.

## [nvim-tree/nvim-web-devicons](https://github.com/nvim-tree/nvim-web-devicons)
- Fork of vim-web-devicons for Neovim, written in lua.

## [nvim-lualine/lualine.nvim](https://github.com/nvim-lualine/lualine.nvim)
- A blazing fast and easy way to configure neovim statusline written in pure lua.

## [SmiteshP/nvim-navic](https://github.com/SmiteshP/nvim-navic)
- A simple statusline/winbar component that uses LSP to show your current code context.

## [akinsho/bufferline.nvim](https://github.com/akinsho/bufferline.nvim)
- A snazzy bufferline for Neovim.

## [mfussenegger/nvim-dap](https://github.com/mfussenegger/nvim-dap)
- Debug Adapter Protocol client implementation.

## [rcarriga/nvim-dap-ui](https://github.com/rcarriga/nvim-dap-ui)
- A UI for nvim-dap which provides a good out of the box configuration.

## [goolord/alpha-nvim](https://github.com/goolord/alpha-nvim)
- The start screen plugin for neovim.

## [akinsho/toggleterm.nvim](https://github.com/akinsho/toggleterm.nvim)
- A Neovim lua plugin to manage terminals in a persistent, smart and intuitive way.

## [b0o/schemastore.nvim](https://github.com/b0o/schemastore.nvim)
- SchemaStore client for Neovim.

## [RRethy/vim-illuminate](https://github.com/RRethy/vim-illuminate)
- Vim plugin for selectively illuminating other uses of the current word under the cursor.

## [lukas-reineke/indent-blankline.nvim](https://github.com/lukas-reineke/indent-blankline.nvim)
- This plugin adds indentation guides to all lines (including empty lines).

## [lunarvim/onedarker.nvim](https://github.com/lunarvim/onedarker.nvim)
- One Darker theme for Neovim.

## [lunarvim/bigfile.nvim](https://github.com/lunarvim/bigfile.nvim)
- Big file optimizations for Neovim.
- Automatically disables certain features if the opened file is big. File size and features to disable are configurable.


This is just a brief overview of the core plugins coming with LunarVim. Click the name of the plugin to learn more!
