# JSON

## Install Syntax Highlighting

```vim
:TSInstall json
```

```lua
{ "jsonls", "rome" }
```

```lua
{ "deno_fmt", "dprint", "fixjson", "jq", "json_tool", "prettier", "prettier_d_slim", "prettierd" }
```

## Supported linters

```lua
{ "cfn_lint", "jsonlint", "spectral" }
```
