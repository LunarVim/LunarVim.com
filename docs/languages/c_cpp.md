# C/C++

## Install Syntax Highlighting

```vim
:TSInstall c
```

```vim
:TSInstall cpp
```

## Supported language servers

```lua
{ "clangd", "ccls" }
```

## Supported formatters

```lua
{ "astyle", "clang_format", "uncrustify" }
```

## Supported linters

```lua
{ "clang_check", "cppcheck", "cpplint", "gccdiag" }
```
