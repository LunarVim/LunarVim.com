# Java

## Install Syntax Highlighting

```vim
:TSInstall java
```

## Supported language servers

```lua
{ "jdtls" }
```

:::note

jdtls requires **jdk-17 or newer** to run.

:::

## Supported formatters

```lua
{ "astyle", "clang_format", "google_java_format", "npm_groovy_lint", "uncrustify" }
```

## Supported linters

```lua
{ "checkstyle", "pmd", "semgrep" }
```
