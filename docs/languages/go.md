# Go

## Install Syntax Highlighting

```vim
:TSInstall go
```

## Supported language servers

```lua
{ "golangci_lint_ls", "gopls" }
```

## Supported formatters

```lua
{ "gofmt", "gofumpt", "goimports", "goimports_reviser", "golines" }
```

## Supported linters

```lua
{ "golangci_lint", "revive", "semgrep", "staticcheck" }
```
