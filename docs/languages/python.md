# Python

## Install Syntax Highlighting

```vim
:TSInstall python
```

## Supported language servers

- jedi_language_server
- pylsp
- pyright
- sourcery

## Supported formatters

- autopep8
- black
- blue
- isort
- reorder_python_imports
- usort
- yapf

## Supported linters

- flake8
- mypy
- pycodestyle
- pydocstyle
- pylama
- pylint
- pyproject_flake8
- semgrep
- vulture
